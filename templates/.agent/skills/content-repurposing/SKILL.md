## 9. Quality Control

### Before Publishing Each Format

| Check | Why |
|-------|-----|
| Platform-native feel? | Looks native, not recycled |
| Right dimensions? | Proper format for platform |
| CTA updated? | Platform-specific action |
| Context added? | Standalone without original |
| Hashtags/Tags? | Platform SEO |

### Common Mistakes

| Mistake | Fix |
|---------|-----|
| Same caption everywhere | Customize for each platform |
| Wrong image ratio | Check specs before posting |
| No context | Content should standalone |
| Too much at once | Space out publishing |
| No tracking | Use UTMs, track performance |
