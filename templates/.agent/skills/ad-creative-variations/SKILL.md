## 2. Platform Specifications

### Google Ads

| Element | Limit | Tips |
|---------|-------|------|
| **Headlines** | 30 chars each (15 max) | Include keywords |
| **Descriptions** | 90 chars each (4 max) | Benefits + CTA |
| **Display URL** | 15 chars each (2 paths) | Relevant keywords |

### Meta Ads (Facebook/Instagram)

| Element | Limit | Tips |
|---------|-------|------|
| **Primary text** | 125 chars (before "See More") | Hook first |
| **Headline** | 40 chars | Benefit-focused |
| **Description** | 30 chars | Supporting info |
| **Image/Video** | Various | 1:1, 4:5, 9:16 |

### TikTok Ads

| Element | Limit | Tips |
|---------|-------|------|
| **Caption** | 100 chars | Hook + CTA |
| **Video** | 9:16 ratio, 5-60s | First 2s crucial |
| **Text overlay** | Readable | Bold, centered |

## 7. Mass Variation Workflow

### Step-by-Step Process

```
1. Define core message/offer
2. Write 5 headline variations (each formula)
3. Write 3 body text variations (each framework)
4. Create 5 CTA variations
5. Combine to create 75+ unique ads
6. Filter top 10-20 for testing
7. Create visual concepts
8. Launch A/B test
```

### Variation Math

| Headlines | x | Body | x | CTA | = Total Combinations |
|-----------|---|------|---|-----|----------------------|
| 5 | x | 3 | x | 5 | = 75 variations |
| 10 | x | 5 | x | 5 | = 250 variations |
